# Point de contrôle

## Installation

Il suffit de récupérer le projet
```
git clone https://github.com/Adrien-Courtois/pcv-projet-thematique.git
```

Installer les requirements python
```
pip install -r requirements.txt
```

Modifier le ficiher de configuration
```
nano .env
```

Lancer le serveur
```
python3 rasp.py
```
